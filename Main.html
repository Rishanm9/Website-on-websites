Websites 101

How to do html and css

The basics: HTML

html is what makes up all the text you see on websites. There's 2 main parts to how a websites looks, html and css. You can think of a website as a drawing. Html makes up the text of a website and other things, it's the sketch of a website. CSS is what adds looks to the website it's just like coloring your drawing.


The basics: CSS

CSS is what makes up how the website looks. It adds a nice touch to your website other than the boring black and white html. It's simple but makes your website a real professional website. You can even make your text pop out like this.


The basics: JavaScript

JavaScript is what gives functions to your website it is writen just like coding languages like python, C, and more. It can animate your website and make your website interactive.
